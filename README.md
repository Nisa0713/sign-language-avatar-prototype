# Sign Language Avatar Prototype

A web-based prototype for visualizing sign language using an animated avatar.

This project demonstrates how written text can be represented through avatar-based sign language animation using frame-based techniques.

## Features

* Frame-based avatar animation system
* Web interface for displaying sign language movements
* Prototype structure designed for future AI/NLP integration

## Technologies

* HTML
* CSS
* JavaScript
* Blender (for avatar animation frames)

## Project Structure

index.html – main web interface
style.css – styling for the interface
app.js – animation and interaction logic

## Note

This repository contains the **prototype structure of the project**.

The full animation frame dataset used in the project is **not included in the public repository**.

Only the application logic and structure are shared for demonstration purposes.

## Future Work

* Integration with Natural Language Processing (NLP)
* Automatic text-to-sign translation
* Expanded avatar animation dataset
